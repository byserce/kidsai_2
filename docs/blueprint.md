# **App Name**: NestableWeb

## Core Features:

- Content Blocks: Enable users to create and manage reusable content blocks.
- Drag-and-Drop Interface: Provide a drag-and-drop interface for arranging content blocks.
- Preview Mode: Allow users to preview the web page as it will appear live.
- AI Content Tool: Generative AI tool to suggest relevant copy for the different content blocks, to populate it in the page.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a sense of sophistication.
- Background color: Light gray (#ECEFF1), nearly white, for a clean and neutral canvas.
- Accent color: Soft purple (#9575CD), which is an analogous color to the indigo primary, but less saturated and lighter.
- Headline font: 'Space Grotesk' sans-serif for headlines; Body font: 'Inter' sans-serif for body; for a clean and modern aesthetic.
- Use simple, geometric icons for a modern feel.
- Maintain a clean and structured layout with clear visual hierarchy.
- Use subtle transitions and animations to enhance the user experience.
